# ChanCMS-admin后台管理

## 介绍

ChanCMS-admin是ChanCMS后台管理界面源码，基于vue3、vue-router、 pinia、element-plus、 mitt、 axios vite4开发的通用的后台管理框架。代码轻量、性能卓越、使用简单。

## 软件架构

```html
    |- vue3
    |- vue-router
    |- pinia
    |- axios
    |- element-plus
    |- mitt
    |- vite4
```

## 安装教程

```javascript
1. npm i
2. npm run dev
3. npm run build
```

## 功能清单

* 1.登录模块&滑块验证码
* 2.接口请求&proxy
* 3.动态&静态菜单&动态面包屑
* 4.基于角色路由权限控制&退出登录&404页面
* 5.ChanCMS后台所有操作功能
* 6.环境配置，dev、sit、prod
* 7.打包配置
